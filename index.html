Panduan Lengkap
1

Generate License Key

Buka menu LICENSE, masukkan User ID Roblox kamu, klik Generate Key. Copy key dan loadstring yang muncul.

Masukkan User ID → Klik Generate → Copy Loadstring
2

Execute Script

Buka executor (Synapse X, Krnl, ScriptWare, dll). Paste loadstring yang sudah di-copy, lalu execute.

loadstring(game:HttpGet("URL_SCRIPT"))()
3

Buka Menu

Setelah execute, akan muncul tombol "Open Menu" di layar. Klik untuk membuka GUI utama.

4

Record Gerakan

Klik "Show Recording UI", tekan REC untuk mulai merekam. Lakukan gerakan, lalu STOP.

5

Save & Play

Klik SAVE untuk menyimpan ke Source Replays. Pilih replay dari dropdown, klik START PLAYBACK.

6

Merge Replay

Pilih minimal 2 replay, klik "+ SELECT TO MERGE", masukkan nama mount, klik MERGE.
